## Start the game

To start our project you need [Node.js](https://nodejs.org/en/download/package-manager/current).

### Install packages

Run on path "./apec4_2048_game/game2048/
```node
npm install
```

This will install all the dependecies from package.json.

### Start game

Run on path "./apec4_2048_game/game2048/
```node
npm start
```

See localhost:3000
